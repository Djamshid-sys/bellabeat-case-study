# Bellabeat Case Study

Google Data Analytics Capstone Project

## Описание
Анализ данных FitBit для разработки стратегии компании Bellabeat.

## Ключевые находки
- 64.9% дней пользователи не достигают 10,000 шагов
- 44.1% ночей недостаточный сон
- Среднее время сидя: 15.9 часов в день

## Инструменты
R, tidyverse, ggplot2, RMarkdown

- Bellabeat_Case_Study.Rmd - код анализа
- Bellabeat_Case_Study.pdf - итоговый отчет
